# 💎 ZYNTH Enterprise IA
### High-Performance Talent Processor v1.0

**ZYNTH** es una solución de software diseñada para automatizar el filtrado de CVs a gran escala. Utiliza modelos avanzados de lenguaje (OpenAI) para analizar, calificar y extraer información crítica de cientos de candidatos en segundos.

## ⚡ Características Principales
* **Procesamiento Masivo:** Análisis de hasta 100+ archivos PDF simultáneamente.
* **Barra de Progreso en Tiempo Real:** Visualización dinámica del estado del análisis.
* **Buscador Inteligente:** Filtro rápido de candidatos por habilidades o puntaje.
* **Seguridad:** Acceso restringido mediante llave maestra (ZYNTH2026).
* **Exportación Directa:** Generación de reportes ejecutivos en formato Excel.

## 🛠️ Stack Tecnológico
* **Lenguaje:** Python 3.10+
* **Framework:** Streamlit (UI/UX)
* **IA:** GPT-4o-Mini / OpenAI API
